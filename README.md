FPGA Trainer Kit

The FPGA Trainer Kit is a hands-on educational platform designed for learning and experimenting with Digital Electronics and FPGA Programming using the Sipeed Tang Nano 9K FPGA board and Verilog HDL.

This trainer kit helps students practically understand digital concepts such as:

Logic Gates
Flip-Flops
Counters
Sequential Circuits
Custom Verilog Designs

The kit includes switch inputs, LED outputs, and a compact hardware setup that allows users to observe digital circuit behavior in real time. Students can write Verilog code, upload it to the FPGA board, and test their designs interactively.

Features :

1.Based on Sipeed Tang Nano 9K FPGA
2.Supports Verilog HDL Programming
3.Real-time LED output indication
4.Interactive switch-based inputs
5.Compact and student-friendly design
6.Ideal for digital electronics learning and experimentation

How It Works :

1.Write the Verilog code
2.Upload the design to the FPGA board
3.Assign switches and LEDs in the code
4.Power the board
5.Test and observe the outputs in real time

Applications :

1.Digital Electronics Labs
2.FPGA Learning
3.Verilog Practice
4.Educational Demonstrations
5.Hardware Prototyping

This project was developed as a Diploma Final Year Project to bridge the gap between theoretical digital electronics and practical FPGA implementation.
